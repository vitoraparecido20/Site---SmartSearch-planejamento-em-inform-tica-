Quem somos

O objetivo do projeto “Smart Search” é desenvolver uma plataforma web inteligente e segura voltada para a busca e localização de pessoas desaparecidas no Brasil. A proposta é utilizar recursos tecnológicos modernos, como bancos de dados integrados, análise de informações e um sistema de busca por dados pessoais (sem o uso de reconhecimento facial), oferecendo uma alternativa eficaz e segura para familiares e autoridades.
